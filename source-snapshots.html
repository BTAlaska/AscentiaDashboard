Publication Policy

Private GitHub source links should not be the only public path for trust signals. Every dashboard-published source claim should have either a public URL that passes link check or a sanitized static snapshot in this deploy mirror.

Run ./check-links.ps1 before deploy. Use ./check-links.ps1 -CheckPublic when network access is available and public HTTP links should be checked too.
